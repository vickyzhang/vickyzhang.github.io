gor 简单使用说明

本来是想答个自己能玩的环境，但是弄了半天都没有弄好，所以听了桥仔的意见，用了我并不知道是什么的gor，使用起来和安装都很方便。

为了我更好的记忆，下次不再让桥仔帮我远程，我还是决定记下来整个使用流程。

首先：
      在下载gor.exe ,并且放在项目目录里面。
      然后把目录写入环境变量。
然后：
      把自己的md文件放入项目目录中的posts文件夹下
最后：
      打开cmd 进入项目目录文件夹下       执行：

    gor http #这样使网站能通过浏览器浏览
    gor compile #这个是编译posts下面的md文件
    gor http #类似于刷新网络的功能
