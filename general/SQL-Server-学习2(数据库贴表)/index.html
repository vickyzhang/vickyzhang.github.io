SQL Server 学习2(数据库贴表)

数据库中一张表的数据 导入 另一张数据表

当两张表的字段完全一致的时候，我们可以直接使用：insert into tab_a select * from tab_b

有字段不同的情况写写法也类似：

INSERT INTO PS_JOBCODE_LANG
(SETID,JOBCODE,LANGUAGE_CD,EFFDT,DESCR,DESCRSHORT,GVT_OFFICIAL_DESCR )  
select SETID,JOBCODE,'ZHS',EFFDT,DESCR,DESCRSHORT,' ' from PS_JOBCODE_TBL
where JOBCODE not in (select JOBCODE from PS_JOBCODE_LANG )

说明：LANGUAGE_CD 中的 LANGUAGE_CD 值固定为’ZHS’ ，而GVT_OFFICIAL_DESCR 的值为空 并且这两个字段不存在于 PS_JOBCODE_TBL 中，所以直接把值填入即可。

（SQL Server 和Mysql 均可使用）
