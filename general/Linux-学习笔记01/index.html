Linux 学习笔记01

  1. Linux 关机命令:

    init 0; shutdown -t 0; shutdown now;

  2. Linux 查看FTP状态是否正常: ps -ef | grep ftp;

    如果没有任何ftp关键字信息，则判断服务是否开启

    service vsftpd status

    开启：service vsftpd start

  3. Linux 查看系统时间 ：date

    设置系统时间：date -s ‘2015-10-27 16:09:01’
